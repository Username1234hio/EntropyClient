Hello guys, to whatever you be seeing this repo for. I made this as a passion project, becuz i love KLOAK. this is simply a temporary version until the offical kloak guys come out with a offical (probably much more better) version.
FUNCTIONALITY 
Quality of life

System tray icon — app minimizes to tray instead of closing, with a right-click menu to open/quit
Remember window size and position between sessions
Startup on login option (launch with Windows/macOS)
Keyboard shortcut to show/hide the window (e.g. Ctrl+Shift+K)

Functionality

Notification badge on the tray icon when you have unread messages (by watching the page title or favicon)
Desktop notifications forwarded from the website
Back/forward navigation buttons in the titlebar
Zoom in/out controls (Ctrl+ / Ctrl-)
Find in page (Ctrl+F)

Developer/power user

Multiple accounts — open a second instance with a different session partition
Custom theme switcher — presets for dark/light/custom CSS stored locally
The CSS injector you already have could save/load named presets

Polish

Splash screen while the webview loads
Proper app icon in taskbar/dock matching the Kloak branding
About page showing version number and update status
